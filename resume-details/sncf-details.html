Data Science Challenge at SNCF

From November 2023 to December 2023

• Built an end-to-end forecasting pipeline in Python (pandas, scikit-learn) to predict next-station occupancy rates from infrared door-count data.

• Achieved a mean absolute error (MAE) of 0.005 versus the 0.01 benchmark, by comparing linear models, Random Forest (best MAE 0.005), neural networks and XGBoost.

• Engineered features including contextual variables (date, train number, station ID, time slot, direction, train composition) and six past occupancy lags per train and per station, handling missing values via KNN imputation and business rules.

• Streamlined data preprocessing: one-hot encoding of categorical fields, datetime feature extraction, correlation-based feature pruning.

• Collaborated via GitHub with branch-based development, pull requests and code reviews to ensure reproducibility and continuous integration.
